# Morpheus - Bootloader

A next-generation UEFI bootloader that enables daily distro-hopping with persistent userland. Boot into a fresh OS every day while keeping your dev environment intact.
This Project shows off deep systems programming and advanced defensive aswell as offensive techniques. The goal of this project is to learn and improve systems engineering capabilities. Morpehus uses some advanced techniques like persistance, obfuscation, self replication aswell as evasion and anti debugging.  


*Dedicated to all the sysadmins who showed me the way <3.*
